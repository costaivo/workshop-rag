# RAG Workshop

This repository contains materials for a **Retrieval-Augmented Generation (RAG)** workshop. You'll learn how to build a complete RAG system that answers questions using your own documents, powered by Python, Google Gemini, and FAISS.

## Getting started

Follow the step-by-step guide to set up your environment and build the system:

**[→ Build Your Own RAG System: Step-by-Step Guide](rag-setup.md)**

The guide walks you through project setup, configuration, document ingestion, and running a working API that queries your documents.
