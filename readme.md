# RAG Workshop

This repository contains materials for a **Retrieval-Augmented Generation (RAG)** workshop. You'll learn how to build a complete RAG system that answers questions using your own documents, powered by Python, Google Gemini, and FAISS—and then add a Streamlit chatbot that talks to your RAG API.

## Workshop flow

1. **RAG system** — Set up the backend: document ingestion, embeddings, FAISS index, and an API that answers questions with sources.
2. **Chatbot** — Build a Streamlit chat interface that calls your RAG API and displays answers with source citations.

## Getting started

Follow the step-by-step guides in order:

1. **[Build Your Own RAG System](rag-setup.md)** — Project setup, configuration, document ingestion, and running a working API that queries your documents.

2. **[Build Your RAG Chatbot](chatbot-setup.md)** — A Streamlit chatbot that talks to your RAG API, with a chat UI and source citations.
